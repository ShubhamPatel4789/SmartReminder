# Smart Reminder App

A fully functional Android reminder app with:
- One-time reminders for any future date/time
- Recurring: Daily / Weekly (pick days) / Monthly (pick day)
- Categories: Health, Payment, Work, Personal
- Toggle reminders on/off
- Survives phone reboots

## How to get the APK (GitHub Actions — Free)

1. Go to **github.com** and create a free account if you don't have one
2. Click **New Repository** → name it `SmartReminder` → Public → Create
3. Upload all these project files (drag & drop the whole folder contents)
4. GitHub Actions will automatically start building
5. Click **Actions tab** → click the latest run → scroll down to **Artifacts**
6. Download **SmartReminder-APK** → unzip → install on your phone

## Install on Samsung Galaxy S23 Ultra

1. On your phone: Settings → Apps → Special Access → Install Unknown Apps
2. Allow your browser/files app to install APKs
3. Open the downloaded APK → Install

## Features
- Tap any reminder to edit it
- Long press to delete
- Toggle switch to enable/disable without deleting
